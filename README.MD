# Loaders Modernized: Integrations

A Space Aged, compact loader the size of a single tile.  These loaders should be relatively UPS friendly
and do not use LUA script to move items.  They also do not use hidden inserters to move inventory.  However, they
do use power.  So keep those power poles handy!

## This mod adds integrations to [Loaders Modernized](https://mods.factorio.com/mod/loaders-modernized) for the following mods

- Krastorio 2
- Krastorio 2 Spaced Out
- Space Exploration
- Bob's Logistics
- Pyanodons
- 5Dim's New Transport
- Ultimate Belts Space Age
- Extended Vanilla: Modular Logistics & Robotics
- More Belts
- Iper Belt
- Wooden Logistics
